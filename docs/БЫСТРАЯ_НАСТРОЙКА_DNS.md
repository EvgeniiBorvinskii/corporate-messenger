## 📝 3 ПРОСТЫХ ШАГА:

### ШАГ 1: Зайдите на сайт регистратора домена

Где вы купили **ypilo.com**? Зайдите туда:
- GoDaddy → godaddy.com
- Namecheap → namecheap.com
- Cloudflare → cloudflare.com
- REG.RU → reg.ru

**Войдите в свой аккаунт.**

---

### ШАГ 2: Найдите настройки DNS

В панели управления найдите ваш домен **ypilo.com** и откройте:
- **DNS Settings**
- **Manage DNS**
- **DNS Management**
- **Управление DNS**

---

### ШАГ 3: Создайте A-запись

Нажмите **Add Record** (Добавить запись) и введите:

```
╔═══════════════════════════════════════╗
║ Type:    A                            ║
║ Name:    @                            ║
║ Value:   165.140.231.35               ║
║ TTL:     3600                         ║
╚═══════════════════════════════════════╝
```

**Нажмите "Save" (Сохранить)**
