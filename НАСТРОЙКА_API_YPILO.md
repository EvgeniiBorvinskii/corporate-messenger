Где вы покупали **ypilo.com**? Зайдите туда:

- REG.RU → reg.ru

---

### ШАГ 2: Добавьте A-запись для поддомена

Нажмите **Add Record** и введите:

```
╔═══════════════════════════════════════╗
║ Type:    A                            ║
║ Name:    api                          ║
║ Value:   5.249.160.54                 ║
║ TTL:     Auto (или 3600)              ║
║ Proxy:   🔴 Выключен (серое облако)   ║
╚═══════════════════════════════════════╝
```

**ВАЖНО:** 
- Name: **api** (не `api.ypilo.com`)
- Value: **5.249.160.54** (IP вашего сервера)
- Proxy: **ВЫКЛЮЧЕН** (облако должно быть серым ☁️)

**Нажмите "Save"**
